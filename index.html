Control de personal

Es una aplicación web desarrollada con JSP de Java y MySQL. Permite llevar el control total del personal de una empresa o institución. Tiene varias funciones como agregar personas, editarlas o eliminarlas del registro.

Venta Online

Es el backend para una página de ventas online. Esta estaba enfocada a un supermercado pero se puede adaptar a cualquier tipo de negocio. Posee las características esenciales de una página de compras online, como el inventario de la mercadería, carrito, control de facturas por usuario, etc.

NodeJS-MongoDB / 2021

Travelink

Consiste en una aplicación en ambiente web, en la se lleva el control de hoteles (tipo trivago), habitaciones, eventos, servicios... Los usuarios pueden reservar las habitaciones disponibles de su hotel favorito. A los administradores les permite ver diferetes estadísticas de mucha ayuda. Así también genera las facturas de cada usuario.

NodeJS-Angular-MongoDB / 2021

Torneos Deportivos

Consiste en una aplicación en ambiente web, la cual consiste en poder llevar el control de un torneo de fútbol. Permite crear las ligas con sus respectivos equipos, el usuario solo debe ingresar los resultados de los enfrentamientos y la aplicación autmáticamente actualizará la tabla de resultados. La aplicación permite generar gráficas y reportes de las ligas.

NodeJS-Angular-MongoDB / 2021
